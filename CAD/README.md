To make the CAD model easier to navigate, the project is provided in two separate files:

### QuattroBox_V2_Assembly_up_to_180mm
Contains the larger parts split into multiple sections, allowing them to be printed on printers with a 180 mm or larger build area.

### QuattroBox_V2_Assembly_up_to_300mm

Contains the larger parts in fewer sections or as full-size parts, intended for printers with a 300 mm or larger build area.
