### Top Mount
For users with Voron printers that have non-standard dimensions, I am providing parametric files compatible with both the Voron and Monolith Panel standards.

Simply open the file in Fusion and modify the frame_size parameter, which represents the total outer length of the frame, measured on the outside and excluding the panels.
